# Kritter v1.0.0

## Highlights

- Added package-based post-format setup flow
- Added reinstall support for common applications through `winget` and direct installers
- Added setup file discovery and packaging support
- Added optimization flows for `Kritter Recommended` and selectable `fr33ty Recommended` scripts
- Added resume-after-restart support for long-running setup workflows
- Added post-install cleanup step

## Release Asset

- `Kritter-v1.0.0-win-x64.zip`

## Requirements

- Windows 10 or Windows 11
- `winget`
- Administrator privileges for some operations
